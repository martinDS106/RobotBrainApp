# 🤖 ROBOT BRAIN - تطبيق التحكم في الروبوت

## 📱 نظرة عامة

تطبيق React Native للتحكم في الروبوتات عن طريق البلوتوث مع واجهة مستخدم سايبربانك.

---

## 🚀 التشغيل السريع

### **طريقة واحدة فقط:**

**اضغط دبل كليك على:**
```
شغّل_التطبيق.bat
```

**هذا الملف يقوم بـ:**
- ✅ إعداد البيئة (JAVA_HOME, ANDROID_HOME)
- ✅ التحقق من Emulator (يشغله لو مش شغال)
- ✅ التحقق من Metro Bundler (يشغله لو مش شغال)
- ✅ تنظيف Gradle Daemons
- ✅ بناء وتثبيت التطبيق
- ✅ فتح التطبيق تلقائياً

---

## 📋 المتطلبات

- **Java 17** (Eclipse Adoptium)
- **Android SDK** (API 24+)
- **Node.js 18+**
- **Android Emulator** أو جهاز حقيقي

---

## 🔧 الإعداد

### 1. تثبيت المتطلبات:

```bash
npm install
```

### 2. تشغيل التطبيق:

**استخدم الملف الجاهز:**
```
شغّل_التطبيق.bat
```

## ⚠️ استكشاف الأخطاء

### التطبيق مش ظاهر في Emulator؟

1. تأكد إن Emulator شغال
2. استخدم `شغّل_التطبيق.bat` تاني
3. افتح App Drawer وابحث عن "RobotBrain"
